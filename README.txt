วิธีใช้
1) อัปโหลดทุกไฟล์และโฟลเดอร์ ยกเว้น appscript ขึ้น GitHub
2) เปิด Google Sheet ใหม่ > Extensions > Apps Script
3) คัดลอก appscript/Code.gs ไปวาง
4) กด Run ฟังก์ชัน setupDatabase() หนึ่งครั้ง
5) Deploy > New deployment > Web app > Anyone
6) คัดลอก URL /exec
7) เปิด js/api.js แล้ววาง URL แทน PASTE_APPS_SCRIPT_EXEC_URL_HERE
8) อัปโหลดขึ้น GitHub อีกครั้ง

ถ้ายังไม่ใส่ URL เว็บจะใช้ข้อมูลตัวอย่างก่อน
